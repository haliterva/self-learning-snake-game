# self-learning-snake-game
Q-Learning algoritması kullanılarak geliştirilen, kendi kendine öğrenebilen Snake oyunu yapay zekası.

Özellikler
- Python ile geliştirildi
- Q-Learning algoritması kullanıldı
- Ajan,ödül-ceza sistemiyle hareket etmeyi öğrendi
- Oyun skoru üzerinden performans takibi yapıldı
- Eğitim sürecinde karar verme mantığı geliştirildi
